# FMCG-Sales-Performance-Analysis
Power BI analysis of FMCG sales data, transforming raw data into interactive dashboards. Explores revenue trends, product performance, and regional insights to support data-driven decisions and business growth.


FMCG Sales Performance Analysis

Project Overview

This project analyzes sales performance data from the Fast-Moving Consumer Goods (FMCG) sector to uncover trends, optimize decision-making, and drive business growth. Using Power BI, I transformed raw sales data into interactive dashboards that provide actionable insights into revenue, product performance, and regional sales distribution.

 Problem Statement

FMCG companies generate large volumes of sales data daily, but often struggle to:
Identify top-performing products and underperforming categories
Track regional sales performance effectively
Monitor revenue trends over time
Make data-driven decisions quickly
This project aims to bridge that gap by building a dynamic and insightful reporting solution.

 Objectives

Analyze overall sales performance and revenue trends
Identify best-selling and low-performing products
Evaluate regional and channel performance
Provide insights to support strategic decision-making

Dataset Description

The dataset includes:
Product categories and SKUs
Sales revenue and quantity sold
Regions and sales channels
Time-based data (monthly/yearly trends)
Note: Dataset is synthetic/hybrid and structured for analytical purposes.

 Tools & Technologies

Power BI (Data visualization & dashboarding)
DAX (Data Analysis Expressions)
Power Query (Data transformation)
Microsoft Excel / CSV (Data source)

 Methodology

1. Data Cleaning & Transformation
Removed duplicates and handled missing values
Standardized data formats (dates, categories, regions)
Created calculated columns for enhanced analysis
2. Data Modeling
Established relationships between tables
Built a star schema for optimal performance
3. DAX Calculations
Key measures created include:
Total Revenue
Total Quantity Sold
Average Sales per Product
Year-to-Date (YTD) Sales
Month-over-Month Growth
4. Dashboard Development
Designed an interactive dashboard with:
KPI cards (Revenue, Sales Volume, Growth Rate)
Trend analysis (line charts)
Regional performance (maps/bar charts)
Product/category breakdown (bar & pie charts)
Filters/slicers for dynamic exploration

 Key Insights

Identified top-performing product categories contributing the highest revenue
Revealed underperforming regions requiring strategic attention
Observed seasonal trends influencing sales performance
Highlighted key drivers of revenue growth


Business Recommendations

Focus marketing efforts on high-performing regions
Improve distribution in low-performing areas
Optimize inventory for fast-moving product
Leverage seasonal trends for campaign planning
